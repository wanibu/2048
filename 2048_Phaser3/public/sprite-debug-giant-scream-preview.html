Giant "Scream" — 巨人大叫/咆哮 3 帧动画

C3 对象 Giant（首页和游戏里顶部那个红发巨人，托着棋盘的角色）的一个动画。
Scream 动画：3 帧，速度 20 fps（50ms/帧），不循环。
每帧尺寸约 424×612，pivot 接近 (0.5, 0.5)（居中）。
来源素材：images/newshared-0-sheet0.png（游戏目前加载的就是新版本）

Giant 全部动画：
- Default（1 帧） — 默认静止（嘴闭，中性）
- Blink（3 帧，28 fps） — 眨眼
- Scream（3 帧，20 fps） — 本页观察对象；观察 3 张连续帧推断是"张嘴咆哮/叫"还是"吃糖"

触发时机：C3 event sheet 里没有用字符串 "Scream" 的调用，推测是通过数值索引切换动画。 可能的触发场景（需在真游戏里对比）：高连击 / 高分合并 / 游戏结束 / 吃掉大糖果 等。

观察点：
• 3 帧是否呈现"嘴先闭 → 张大 → 再收"的节奏？→ 很可能是"叫/咆哮"
• 3 帧是否呈现"空手 → 手中或嘴里有糖果 → 吞下"？→ 是"吃糖"
• 3 帧是否呈现其他变化（例如挤眉弄眼、脸变红）？→ 是其他表情
