Combo 链条 — Phaser 4 实现版

用 Phaser 4（CDN）重写 CSS 版本的相同功能。用于验证 Phaser 实际渲染效果与 CSS 预览是否一致。

Phaser 4 vs CSS 版对比

• 一致：参数默认值、触发时序、动画节奏
• 差异：Phaser 用真 Tween/Anims 引擎（含 GPU 加速），CSS 用 transition/setInterval
• Phaser 优势：easing 更精细、批量 sprite 性能更好、可导出为游戏资产
