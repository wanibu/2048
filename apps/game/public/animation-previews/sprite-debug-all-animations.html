原版全部动画 — 综合分析

从 Construct 3 data.json（与 mini-game.pwtk.cc 线上版一致）反推。
本页展示所有带帧素材的对象，按类型分类：真动画（时间自动播）、状态帧（代码手动切）、静态图标（单帧）
并列出每个对象的 行为（Tween/Fade/Bullet/Pin/Anchor）、container 伙伴、所用 sprite sheet。
Tween 属性平滑过渡（x/y/scale/rotate/alpha）
Fade 淡入淡出（alpha）
Bullet 弹道运动（初速+重力）
Pin 钉在另一对象上
Anchor 屏幕锚定（UI 对齐边缘）
