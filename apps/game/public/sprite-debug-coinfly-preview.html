CoinPlay + Fly — 金币飞行模拟

CoinPlay（50×50 金币图标，单帧静态）与 Fly（C3 行为）在同一 container。
金币飞行动画不是帧动画，而是 CoinPlay 静态图标通过 Fly 行为沿路径从棋盘位置运动到右上角金币 UI 的过程。
本页用 JS 模拟这个飞行：起点在棋盘中心，终点在右上角金币 UI；可选直线/弧线路径。

实现建议（集成到 Phaser 时）：
• 用 scene.add.image(startX, startY, 'shared3', 'coin-icon-frame') 创建金币
• 用 scene.tweens.add({ targets, x: endX, y: endY, duration, ease: 'Cubic.easeIn' }) 做弧线
• 结束时 destroy() 并同时 hud.addCoin(1)
