Phaser 4 实现要点：
• texture.add() 注册 Shape 4 帧数字 + Giant Default + BorderExplode 11 帧
• 每个数字起始 Phaser.GameObjects.Image 位置 (startX, 1240)
• this.tweens.add({ y, duration, ease }) 飞起 + 回弹
• time.delayedCall 串起 stagger 和 burst 延迟
• BorderExplodeAnimation 用 time.addEvent loop 手动切 texture frame
